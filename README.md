# Full House — лендинг барбершопа

Одностраничный лендинг для мужской парикмахерской **Full House** (Новосибирск, Горский
микрорайон). Цель страницы — запись клиента: форма, кликабельный телефон и мессенджеры.
Кейс для портфолио веб-разработчика.

🔗 Демо: https://nikita41478.github.io/fullhouse/

## Стек

Чистый HTML / CSS / JS, без фреймворков и сборщиков. Вся страница — один файл `index.html`
(разметка + `<style>` + `<script>` IIFE).

- Шрифты — Google Fonts (**Forum** для заголовков + **Commissioner** для текста, с кириллицей).
- Иконки — инлайн-SVG (без внешних библиотек и иконочных шрифтов).
- Карта в контактах — встроенный iframe Яндекс-Карт, грузится лениво (`loading="lazy"`).
- Фото работ, мастеров, услуг и интерьера — локальные, в папке `pictures/` (оптимизированы в `.webp`).
- Hero — анимация падающих игральных карт (лёгкая физика на `requestAnimationFrame`, только
  `transform`/`opacity`; ставится на паузу вне экрана и уважает `prefers-reduced-motion`).
- Анимации появления — на IntersectionObserver. Без тяжёлых библиотек и автоплея.

## Структура

| Файл / папка | Назначение |
|---|---|
| `index.html` | Вся страница: hero, работы, услуги, о барбершопе и мастера, отзывы, FAQ, запись, контакты. |
| `pictures/` | Фото работ (`Works_*.webp`), мастеров (`master_*.webp`), услуг (`service_*.webp`), интерьера (`office_*.webp`), логотип (`logo.png`) и og-картинка (`og.jpg`). |
| `.nojekyll` | Отключает обработку Jekyll на GitHub Pages (статика отдаётся как есть). |

## Локальный запуск

Это статический сайт, сборка не нужна. Достаточно поднять любой локальный сервер:

```bash
# через Node
npx --yes serve -l 4321 .
# затем открыть http://localhost:4321
```

```bash
# или Python 3
python -m http.server 4321
```

Или просто открыть `index.html` в браузере (карта и шрифты требуют интернета).

## Деплой

Сайт опубликован на GitHub Pages из ветки `main` (корень репозитория). Любой пуш в `main`
автоматически обновляет страницу.
